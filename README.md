# XYCLIAE (Unity3D)

Xycliae is an experimental macOS application inspired by John Whitney's Music Box

## Usage
#### Sequence Modes
0: Random,  1: Index A
2: Index B,  3: Rotate A
4: Rotate B,  5: Skip

#### Pitch Modes
0: None,  1: Index A
2: Index B,  3: Rotate A
4: Rotate B,  5: Random

#### Shortcuts
Ctrl+F: Toggle fullscreen
F: Toggle UI
T: Toggle particle trails
